# nih-chestxray-data-exploration
# NIH ChestXray14 Dataset Exploration & Preprocessing

This project focuses on exploratory data analysis (EDA) and preprocessing of the NIH ChestXray14 dataset for multi-label classification tasks.

The notebook covers dataset understanding, label distribution, missing value analysis, and preparation of training-ready metadata.

---

## Key Highlights
- Exploratory data analysis on NIH ChestXray14 metadata
- Multi-label frequency and label distribution analysis
- Missing value analysis and data consistency checks
- Preparation of structured CSV labels for training pipelines

---

## Dataset
- NIH ChestXray14 dataset
- Metadata and labels in CSV format

> Note: Dataset files are not uploaded to GitHub due to size restrictions.

---

## How to Run (Google Colab)
1. Open the notebook from `notebooks/`
2. Mount Google Drive (if dataset is stored in Drive)
